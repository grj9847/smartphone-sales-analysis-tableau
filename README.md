# 📊 Smartphone Sales Analysis Dashboard (Tableau)

## 📌 Overview
This project analyzes smartphone sales data to uncover insights on revenue, pricing, discounts, and brand performance using an interactive Tableau dashboard.

## 🛠 Tools Used
- Tableau
- Excel (Data Source)

## 📈 Key Features
- KPI metrics including Total Revenue, Total Discount, Average Rating, Average Price, and Total Phones Sold
- Brand-wise analysis of smartphone models and performance
- Price vs Rating analysis to understand customer perception
- Discount analysis across brands
- Price distribution and variation analysis
- Interactive dashboard for dynamic filtering and exploration

## 🔍 Key Insights
- Identified top-performing brands based on number of models and revenue
- Analyzed pricing strategies across different smartphone brands
- Observed relationship between price and customer ratings
- Evaluated discount patterns and their impact on sales
- Highlighted variations in pricing across brands and segments

## 📁 Files Included
- Tableau Dashboard (.twbx / .twb)
- Dataset (if available)
